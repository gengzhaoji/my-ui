Three.js - 添加天空盒 #

天空盒 #

  • 在 3D 场景中，为了增强场景表现力，我们通常会为场景添加天空盒。就是在整个场景的上方绘制天空这一类图片。
  • 最简单的做法是，制作一个很大的立方体为 6 个面添加纹理，这里要注意我们渲染的是内部。纹理是特殊处理了的能组合成一个整体。当然还有其他的方式比如创建球体或半圆的穹顶等。

创建天空盒的几种方法 #

TIP

常用的为创建盒子 和 设置 scene 的背景

1、创建盒子 #

// 在场景中加入天空盒
{
    const directions = ['px.jpg', 'nx.jpg', 'py.jpg', 'ny.jpg', 'pz.jpg', 'nz.jpg'], // 获取对象
        TextureLoader = new THREE.TextureLoader(),
        // 创建盒子，并设置盒子的大小为
        skyGeometry = new THREE.BoxGeometry(700, 700, 700),
        // 设置盒子材质
        materialArray = [];
    for (let i = 0; i < 6; i++) {
        materialArray.push(
            new THREE.MeshBasicMaterial({
                map: TextureLoader.load(`/models/demo1/${directions[i]}`), // 将图片纹理贴上
                side: THREE.DoubleSide, // 需要注意材质要设置`THREE.BackSide`渲染内部，或者设置`THREE.DoubleSide`渲染双面，相机的远平面必须大于盒子的长度
            })
        );
    }
    // 创建一个完整的天空盒，填入几何模型和材质的参数
    const skyBox = new THREE.Mesh(skyGeometry, materialArray);
    scene.add(skyBox);
}

TIP

注意 directions[]数组中的图片顺序是有要求而不是随意的！否则加载的效果会错乱。这张图片会帮助你更好的理解。

2、利用 CubeTextureLoader 设置 scene 的背景 #

const path = 'textures/cube/Park3Med/', //设置路径
    format = '.jpg', //设定格式
    urls = [path + 'px' + format, path + 'nx' + format, path + 'py' + format, path + 'ny' + format, path + 'pz' + format, path + 'nz' + format],
    textureCube = new THREE.CubeTextureLoader().load(urls);

scene.background = textureCube; //作为背景贴图

TIP

更加简洁的写法:

scene.background = new THREE.CubeTextureLoader().setPath('textures/cube/Park3Med/').load(['px.jpg', 'nx.jpg', 'py.jpg', 'ny.jpg', 'pz.jpg', 'nz.jpg']);

3、360 度的全景图 #

使用 WebGLCubeRenderTarget 渲染器的 .fromEquirectangularTexture，把全景图转换为立方体贴图格式。

{
    const loader = new THREE.TextureLoader();
    const texture = loader.load('./file/18/2.webp', () => {
        const rt = new THREE.WebGLCubeRenderTarget(texture.image.height);
        rt.fromEquirectangularTexture(renderer, texture);
        scene.background = rt.texture;
    });
}
