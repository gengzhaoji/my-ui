THREEJS OrbitControls 核心源码分析讲解 #

THREE.js 的 OrbitControls 控制器是一个用于相机交互控制的类。通过分析其核心源码，可以深入了解其实现原理。OrbitControls 主要通过事件监听器捕捉鼠标和键盘的交互行为，并对相机进行变换，提供旋转、缩放和平移功能。

下面是对 OrbitControls 核心源码的详细分析和讲解。

1. 基本属性初始化 #

OrbitControls 初始化时，会设置相机、目标元素（通常是 canvas），以及控制各种功能的参数。

constructor( camera, domElement ) {
  this.camera = camera;
  this.domElement = domElement;

  // 初始状态
  this.target = new THREE.Vector3();  // 相机围绕的目标点
  this.enabled = true;  // 是否启用控制器
  this.enableRotate = true;  // 是否允许旋转
  this.enableZoom = true;  // 是否允许缩放
  this.enablePan = true;  // 是否允许平移

  // 阻尼（惯性）控制
  this.enableDamping = false;
  this.dampingFactor = 0.05;

  // 缩放控制
  this.minDistance = 0;  // 最小距离
  this.maxDistance = Infinity;  // 最大距离
  this.zoomSpeed = 1.0;

  // 旋转控制
  this.rotateSpeed = 1.0;
  this.minPolarAngle = 0;  // 上下旋转的最小角度
  this.maxPolarAngle = Math.PI;  // 上下旋转的最大角度
  this.minAzimuthAngle = -Infinity;  // 水平方向旋转最小角度
  this.maxAzimuthAngle = Infinity;  // 水平方向旋转最大角度

  // 平移控制
  this.keyPanSpeed = 7.0;  // 键盘平移速度
  this.autoRotate = false;  // 是否自动旋转
  this.autoRotateSpeed = 2.0;
}

说明：

  • camera: 控制的相机对象。
  • domElement: 绑定的 HTML 元素，通常是 canvas。
  • target: 相机围绕的目标点（通常是场景中心）。
  • enableRotate、enableZoom、enablePan: 是否启用旋转、缩放和平移功能的开关。
  • dampingFactor: 阻尼因子，控制旋转、缩放和平移时的惯性效果。

2. 事件监听和处理 #

OrbitControls 通过鼠标、触摸屏和键盘事件来捕捉用户的交互操作。核心事件包括：

  • 鼠标按下（mousedown）事件：开始旋转、平移或缩放操作。
  • 鼠标移动（mousemove）事件：根据鼠标移动计算旋转、平移或缩放的变换量。
  • 鼠标滚轮（wheel）事件：执行缩放操作。
  • 鼠标松开（mouseup）事件：结束操作。
this.domElement.addEventListener('mousedown', this.onMouseDown.bind(this), false);
this.domElement.addEventListener('mousemove', this.onMouseMove.bind(this), false);
this.domElement.addEventListener('mouseup', this.onMouseUp.bind(this), false);
this.domElement.addEventListener('wheel', this.onMouseWheel.bind(this), false);

核心事件处理函数：

  • onMouseDown: 处理鼠标按下事件，决定旋转、缩放或平移。
  • onMouseMove: 处理鼠标拖动事件，更新相机视角。
  • onMouseWheel: 处理缩放（滚轮事件）操作。

3. 旋转功能实现 #

旋转是通过改变相机的位置，使其绕着 target 进行球面轨迹运动。旋转功能实现了绕目标点的俯仰（polar angle）和水平（azimuth angle）两个方向的旋转。

rotateLeft( angle ) {
  sphericalDelta.theta -= angle;
}

rotateUp( angle ) {
  sphericalDelta.phi -= angle;
}
  • rotateLeft：通过调整 sphericalDelta.theta 值来控制水平方向的旋转。
  • rotateUp：通过调整 sphericalDelta.phi 来控制垂直方向的旋转。

每次 mousemove 事件都会触发 rotateLeft 和 rotateUp 函数，根据鼠标移动的距离计算旋转角度：

function onMouseMove(event) {
    if (isRotating) {
        rotateLeft(((2 * Math.PI * mouseDelta.x) / domElement.clientWidth) * rotateSpeed);
        rotateUp(((2 * Math.PI * mouseDelta.y) / domElement.clientHeight) * rotateSpeed);
    }
}

4. 缩放功能实现 #

缩放操作通过改变相机与目标点之间的距离来实现。相机沿着视线方向前后移动。

dollyIn( dollyScale ) {
  scale /= dollyScale;
}

dollyOut( dollyScale ) {
  scale *= dollyScale;
}
  • dollyIn：相机向目标点靠近，缩小视角。
  • dollyOut：相机远离目标点，放大视角。

在 mousewheel 事件中，根据滚轮的方向来判断是调用 dollyIn 还是 dollyOut：

function onMouseWheel(event) {
    if (event.deltaY < 0) {
        dollyIn(getZoomScale());
    } else if (event.deltaY > 0) {
        dollyOut(getZoomScale());
    }
}

5. 平移功能实现 #

平移操作是通过改变 target 的位置实现的，即相机的位置保持不变，但目标点在世界坐标系中的位置发生变化。

panLeft( distance, objectMatrix ) {
  var v = new THREE.Vector3();
  v.setFromMatrixColumn( objectMatrix, 0 );  // get X column of objectMatrix
  v.multiplyScalar( - distance );
  panOffset.add( v );
}

panUp( distance, objectMatrix ) {
  var v = new THREE.Vector3();
  v.setFromMatrixColumn( objectMatrix, 1 );  // get Y column of objectMatrix
  v.multiplyScalar( distance );
  panOffset.add( v );
}
  • panLeft：在水平方向平移目标点。
  • panUp：在垂直方向平移目标点。

6. 阻尼（惯性）效果 #

阻尼效果是通过在每次渲染时，逐步减小旋转、缩放或平移的速率来实现的，提供一种平滑的拖动效果。

if (this.enableDamping === true) {
    sphericalDelta.theta *= 1 - this.dampingFactor;
    sphericalDelta.phi *= 1 - this.dampingFactor;
    scale *= 1 - this.dampingFactor;
    panOffset.multiplyScalar(1 - this.dampingFactor);
}

每帧渲染时，sphericalDelta 和 panOffset 的值都会根据阻尼系数逐渐减小，模拟惯性减速效果。

7. 更新相机位置 #

每次交互（如旋转、缩放或平移）后，都会调用 update 方法来更新相机的位置。

update() {
  var offset = new THREE.Vector3();
  offset.copy( this.camera.position ).sub( this.target ); // 获取相机与目标点的相对位置
  offset.applyQuaternion( quat ); // 旋转相机
  this.camera.position.copy( this.target ).add( offset ); // 更新相机位置
  this.camera.lookAt( this.target );  // 让相机始终指向目标点
}

总结：

  • OrbitControls 通过鼠标和键盘事件监听用户的交互行为，并根据相应的操作（旋转、缩放、平移）来更新相机位置。
  • 核心功能如旋转、缩放和平移是通过计算球面坐标（sphericalDelta）和相机与目标点之间的相对位移实现的。
  • 阻尼效果提供了平滑的用户体验，而自动旋转、限制旋转角度、缩放范围等功能则提升了相机控制的灵活性和安全性。
