vite 打包图片压缩 #

vite-plugin-imagemin 和 vite-plugin-image-optimizer 都是用于 Vite 项目图片压缩优化的插件，但它们在实现方式、功能和依赖项上有显著差异。以下是详细对比：

1. 核心实现与依赖 #

特性vite-plugin-imageminvite-plugin-image-optimizer
底层工具基于 imagemin 及其插件生态（如 mozjpeg, pngquant）基于 sharp（高性能图像处理库）或原生 squoosh
依赖项需要额外安装 imagemin-* 插件（如 imagemin-pngquant）仅需 sharp（默认推荐）或零依赖（使用 squoosh）
安装复杂度较高（需手动选择并安装插件）较低（开箱即用）

2. 功能与压缩效果 #

特性vite-plugin-imageminvite-plugin-image-optimizer
压缩算法支持多格式插件（如 mozjpeg/pngquant/svgo）使用 sharp 的优化算法（自动格式适配）
格式支持需手动配置各格式插件自动处理常见格式（JPG/PNG/WebP/AVIF/SVG）
压缩率控制高度可定制（每个插件独立配置）简单参数配置（如 quality）
WebP/AVIF 转换需手动配置插件（如 imagemin-webp）内置支持（通过 sharp）

3. 性能与构建速度 #

特性vite-plugin-imageminvite-plugin-image-optimizer
速度较慢（链式调用多个插件）极快（sharp 基于 C++ 原生性能）
缓存机制无内置缓存支持缓存（避免重复压缩）
并行处理依赖 Node.js 线程池自动并行化（sharp 优化）

4. 配置复杂度 #

vite-plugin-imagemin 示例 #

// vite.config.js
import imagemin from 'vite-plugin-imagemin';

export default {
    plugins: [
        imagemin({
            gifsicle: { optimizationLevel: 7 },
            mozjpeg: { quality: 20 },
            pngquant: { quality: [0.8, 0.9] },
            svgo: {
                plugins: [{ name: 'removeViewBox' }, { name: 'cleanupIDs' }],
            },
        }),
    ],
};

vite-plugin-image-optimizer 示例 #

// vite.config.js
import { ViteImageOptimizer } from 'vite-plugin-image-optimizer';

export default {
    plugins: [
        ViteImageOptimizer({
            jpg: { quality: 80 },
            png: { quality: 80 },
            webp: { lossless: true },
        }),
    ],
};

5. 适用场景 #

场景推荐插件
需要深度定制压缩参数vite-plugin-imagemin
追求极简配置和快速构建vite-plugin-image-optimizer
需要 WebP/AVIF 自动转换vite-plugin-image-optimizer
老旧项目兼容性要求vite-plugin-imagemin

6. 其他注意事项 #

  • vite-plugin-imagemin 的痛点:
    • 依赖链复杂（需手动安装 imagemin 插件）
    • 可能因 Node.js 版本或系统环境导致安装失败（如 imagemin 的二进制依赖）
  • vite-plugin-image-optimizer 的优势:
    • 零配置默认优化（只需安装插件）
    • 支持现代格式（如 AVIF）和响应式图片生成
    • 与 Vite 的按需加载机制深度集成

总结建议 #

  • 如果项目需要 精细控制压缩参数 或兼容老旧工具链，选择 vite-plugin-imagemin。
  • 如果追求 开箱即用、高性能和现代格式支持，优先选择 vite-plugin-image-optimizer。

两种插件均可通过以下命令快速安装：

# 安装 imagemin 版本
npm install vite-plugin-imagemin imagemin-mozjpeg imagemin-pngquant imagemin-svgo

# 安装 image-optimizer 版本（推荐）
npm install vite-plugin-image-optimizer sharp
