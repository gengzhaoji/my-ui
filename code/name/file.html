语言规范编码规范

确保文件命名总是以字母开头而不是数字，且字母一律小写，以下划线连接且不带其他标点符号，如：

<!-- HTML -->
rdc.html
rdc_list.html
rdc_detail.html

<!-- SASS -->
rdc.scss
rdc_list.scss
rdc_detail.scss
