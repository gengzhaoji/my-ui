润道前端 #

润道前端基础后台项目模板下载

方法 1 #

git clone http://192.168.1.16:9980/fe/my-ui

修改git地址，进入项目目录安装项目依赖（建议使用pnpm包管理工具pnpm包管理快速、简洁)。

pnpm i

pnpm run dev

方法 2 #

npm i gz-vue-cli -g

gz init [项目名]

选择vue3 + element基础模板,随后选择包管理安装项目依赖

生成 pai #

npm run coder (api[生成pai]/icon[生成icon])
